# Coarev

The name Coarev comes from basic psichological process of memory that I found in a portuguese [paper](https://cursoestudomemorizacao.com.br/memoria-psicologia/):
- Codificação (encoding)
- Armazenamento (storage)
- Evocação (evocation)
